# Collectible data points for Health Monitoring System

## From node

- CPU usage
- RAM usage
- CPU temperature
- Voltage used
- Current used
- Wattage used 
- Docker service status
  - Number of docker containers running
- IP address
- Hostname
- Storage remaining
- OS Version

## From admin

- Cluster status
- CassandraDB status
- Kafka status
- Voltage used by lights
- Current used by lights
- Wattage used by lights
- Wattage used by switch
